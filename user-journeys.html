Designing user journeys and conversation strings for conversational assistants

context

Finn AI partners with other companies to expand and improve its product offering to banks. For example, we partnered with Visa Canada to build a conversational credit card manager for banks. To secure partnerships, Finn AI creates conceptual conversations to showcase potential use cases.

action

I was tasked with creating some conceptual use cases. This involved designing a user journey that guides users, eliminates bottlenecks, and is intuitive just like a real conversation. I documented user journeys in an easy-to-follow flow chart. I learned to design conversation strings by shadowing a conversation designer and studying Google's guidelines for conversation design.

results

The user journey and conversation strings I wrote were included in the demo used to secure Visa Canada as a partner.
